# Images Directory

## Background Images

### Hero Background (hero-bg.jpg)
Place your school's hero/banner background image here as `hero-bg.jpg`

Recommended specifications:
- Format: JPG or PNG
- Dimensions: 1920x1080px or higher
- Size: Under 500KB (optimized for web)
- Content: School campus, building, or graduation ceremony

## Other Images
You can store other website images in this directory or create subdirectories for organization.
